## 1. 工业极简主义 (Industrial Minimalism)

### Core Principles
- **功能至上**: 界面元素服务于数据展示和操作效率，无多余装饰。
- **高对比度**: 使用深色背景和亮色数据点，模拟工业控制台的视觉体验。
- **实时反馈**: 操作和状态变化必须有即时的视觉反馈（如指示灯、进度条）。
- **模块化布局**: 像仪表盘一样，每个功能区独立且清晰。

### Color Philosophy
- **深空灰背景**: 减少长时间注视的视觉疲劳，营造专业感。
- **霓虹蓝/绿点缀**: 用于状态指示和关键操作，代表科技与活力。
- **警示红/橙**: 仅用于错误和警告，确保异常状态醒目。

### Layout Paradigm
- **仪表盘式**: 左侧导航，右侧多面板布局。
- **卡片化**: 每个设备或命令组作为一个独立的卡片，便于信息聚合。

### Signature Elements
- **单色图标**: 简洁的线条图标，易于识别。
- **数据可视化**: 使用迷你图表（Sparklines）展示CPU/内存趋势。
- **状态指示灯**: 经典的红绿圆点，直观显示在线/离线状态。

### Interaction Philosophy
- **点击即响应**: 按钮点击有明确的按下效果。
- **悬停揭示**: 详细信息（如完整ID、时间戳）在鼠标悬停时显示，保持界面整洁。

### Animation
- **微动效**: 状态切换时的平滑过渡，避免突兀跳变。
- **加载指示**: 细微的进度条或旋转图标，不打断用户操作流。

### Typography System
- **等宽字体**: 用于ID、代码和日志，确保对齐和可读性（如 JetBrains Mono）。
- **无衬线字体**: 用于标题和标签，清晰现代（如 Inter 或 Roboto）。

## 2. 赛博朋克控制台 (Cyberpunk Console)

### Core Principles
- **未来感**: 模拟科幻电影中的黑客终端。
- **信息密集**: 允许在一个屏幕上展示大量数据。
- **故障美学**: 偶尔的Glitch效果，增加沉浸感。

### Color Philosophy
- **黑底**: 纯黑背景。
- **高饱和度**: 亮紫、青色、洋红，形成强烈的视觉冲击。

### Layout Paradigm
- **网格系统**: 严格的网格对齐，带有可见的网格线。
- **HUD风格**: 悬浮层和半透明背景。

### Signature Elements
- **发光边框**: 选中元素带有外发光。
- **扫描线**: 背景带有淡淡的扫描线纹理。
- **终端字体**: 像素风格或复古计算机字体。

### Interaction Philosophy
- **键盘优先**: 支持快捷键操作。
- **声音反馈**: (可选) 操作时的机械音效。

### Animation
- **打字机效果**: 文本逐字显示。
- **Glitch**: 切换页面时的短暂故障效果。

### Typography System
- **像素字体**: 标题使用像素风格。
- **终端字体**: 正文使用复古终端字体。

## 3. 清新办公风 (Clean Office)

### Core Principles
- **亲和力**: 类似Notion或Airtable的界面，易于上手。
- **明亮**: 白色或浅灰背景，减少压抑感。
- **柔和**: 圆角和阴影，增加层次感。

### Color Philosophy
- **白色/浅灰**: 主色调。
- **柔和蓝/紫**: 品牌色，用于强调。

### Layout Paradigm
- **列表视图**: 类似表格的布局，适合大量数据浏览。
- **侧边栏**: 传统的侧边栏导航。

### Signature Elements
- **大圆角**: 卡片和按钮使用大圆角。
- **柔和阴影**: 增加深度。
- **多彩标签**: 状态标签使用柔和的背景色。

### Interaction Philosophy
- **拖拽**: 支持拖拽排序或分组。
- **模态框**: 详情在模态框中打开。

### Animation
- **淡入淡出**: 页面切换柔和过渡。
- **缩放**: 按钮点击时的轻微缩放。

### Typography System
- **圆体**: 标题使用圆润的字体。
- **系统字体**: 正文使用系统默认无衬线字体。

## 选择方案：1. 工业极简主义 (Industrial Minimalism)

**理由**:
- **专业性**: 设备控制系统需要给人稳定、可靠的感觉，深色工业风最符合这一需求。
- **清晰度**: 高对比度和模块化布局有利于快速识别设备状态和异常。
- **效率**: 减少装饰，专注于数据和操作，符合工具类软件的定位。
- **技术感**: 与底层gRPC、Linux、Docker等技术栈的气质相符。

**设计执行**:
- **背景**: 深灰 (`#1a1a1a`)。
- **强调色**: 科技蓝 (`#3b82f6`) 和 信号绿 (`#10b981`)。
- **字体**: 标题使用 `Inter`，代码/ID使用 `JetBrains Mono`。
- **组件**: 使用 Shadcn UI 的深色模式，配合自定义的边框和阴影。
